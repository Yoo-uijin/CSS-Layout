Flexbox

1. Rule

flexbox를 이용하여 box들의 위치를 바꾸고 싶다면 box(자식)가 아닌 flex container(부모)에게 display: flex;
단, 바로 붙어있는 부모만 자식의 위치를 움직일 수 있다.

2. main axis & cross axis

flex-direction: row(default)일 때,
main axis => horizontal axis
cross axis => vertical axis

justify-content: main axis 방향으로 item을 옮긴다.
align-item: cross axis 방향으로 item을 옮긴다.
주의! align-item: center; 을 했는데 item이 가운데로 이동하지 않는다면, flex container(부모)의 height를 확인 할 것! item과 container의 높이가 같은 경우일 가능성 높음 => .container {height: 50vh;}
